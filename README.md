# CPP Module 01 42 Madrid

# Installation
- Clone the repo.
```bash
git clone https://github.com/danielfdez17/cpp-module-01.git
cd cpp-module-01
```
# Build
Every exercise is inside a folder name ex0n, where n is a number greater than 0. Each folder has its own Makefile.

If you have make installed in your system, just build the program with make. Otherwise, you will need to install it.
```bash
make
```

# Usage
There is a Makefile rule to run/test every program.
```bash
make run || make tests
```
